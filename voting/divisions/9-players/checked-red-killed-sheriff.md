# Проверенный красный у ПУ шерифа: деление в сторону

## Позиция

|  |  |
| :--- | :--- |
| Я | игрок 9, красный |
| Умер | игрок 10 — шериф с проверкой «9 — красный» |
| Красные цвета | 8 (по наблюдениям дня 1) |
| Связки «не два чёрных» | {1, 2}, {3, 4} |

![](seats.svg)

## Возможные проблемы

{% hint style="warning" %}
**Деление трёх красных на 3в3**
{% endhint %}

{% hint style="warning" %}
**Деление без плана на игру после подъёма**
{% endhint %}

## Что решает проблему

{% hint style="info" %}
Я и 8 — красные → в {1–7} остаётся **4 красных и 3 чёрных**. Каждая связка уносит максимум одного чёрного, обе вместе — максимум двух → **минимум один чёрный сидит в {5, 6, 7}**.
{% endhint %}

{% hint style="success" %}
**Деление 5, 6, 7.** При исходных допущениях (8 — красный, обе связки валидны) гарантированно попадаю минимум в одного чёрного. Более того, при несломе на круг при пятерых будет оставлен красный цвет и две связки, которые уменьшают комбинаторную сложность игры
{% endhint %}
